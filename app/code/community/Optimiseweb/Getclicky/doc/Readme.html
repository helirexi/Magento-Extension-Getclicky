Clicky (new name for GetClicky) is a real time web analytics service. This extension is a Clicky integration for the Magento eCommerce platform. Optimise Web's Clicky integration extension allows store owners to enable Clicky tracking for their websites. This extension can send checkout successes and revenues as dynamic goals to Clicky.

Features

Latest Clicky Code
- Uses up-to-date Clicky code.
- Supports 'https' (secure URLs).
- Switch between 'Default', 'Asynchronous', and 'Non-Javascript' from the system configuration panel.

Fully Configurable
- Using the system configuration panel. System > Configuration > Optimise Web (tab) > Clicky Integration

FAQs

1. Why am I not seeing the system settings? Why am I getting a 404 error?

As with the installation of all Magento modules, please clear the cache folders and log out of the admin area and log back in. Please also flush your CSS and JS caches.

Yes. Please make it a point to backup your files and database before installing or upgrading any Magento module.
